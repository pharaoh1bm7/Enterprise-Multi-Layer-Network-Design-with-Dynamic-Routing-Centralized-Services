# 🧠 Enterprise Multi-Layer Network Design

This project demonstrates the design and implementation of a scalable enterprise network using a multi-layer architecture.

## 📌 Overview

The network is designed using the three-tier architecture model:

- Core Layer
- Distribution Layer
- Access Layer

The goal of this project is to simulate a real-world enterprise environment with full connectivity between all devices including servers, PCs, laptops, and wireless clients.

---

## 🏗️ Network Architecture

- **Core Layer**: Routers connected together for WAN communication  
- **Distribution Layer**: Multilayer Switch (Layer 3) handling routing between VLANs  
- **Access Layer**: Switches connecting end devices (PCs & Laptops)  

---

## ⚙️ Key Features

- ✅ VLAN Segmentation for network organization  
- ✅ Inter-VLAN Routing using Layer 3 Switch  
- ✅ Dynamic Routing between routers  
- ✅ Centralized Servers accessible from all networks  
- ✅ Wireless connectivity using Access Point  
- ✅ Full communication between all devices
