# Enterprise-Multi-Layer-Network-Design-with-Dynamic-Routing-Centralized-Services

<img width="1793" height="816" alt="Screenshot 2026-03-31 020556" src="https://github.com/user-attachments/assets/afb2239f-2275-4587-b6ba-83e475dc0e23" />

This project demonstrates the design and implementation of a scalable enterprise network using a multi-layer architecture.

The network is built using core, distribution, and access layers to ensure high performance, scalability, and efficient traffic management. A multilayer switch is used to perform inter-VLAN routing, enabling communication between different network segments.

Key features of the project include:

Implementation of VLAN segmentation to isolate network traffic
Inter-VLAN routing using a Layer 3 switch
Dynamic routing configuration between routers for seamless connectivity
Integration of centralized servers accessible from all network segments
Support for both wired and wireless devices through access switches and access points
End-to-end connectivity across all departments and devices

The project simulates a real-world enterprise environment where multiple departments, servers, and wireless clients communicate efficiently within a unified network infrastructure.
